### v3 问题修复（2026-06-03 第三次修改）

#### 问题1: 目录和内容独立滚动
**原因**: `.epub-layout` 没设固定高度，整个面板一起滚
- [x] `.epub-layout` 增加 `max-height: calc(70vh - 88px)` + `overflow: hidden`
- [x] 两侧各自 `overflow-y: auto` 独立生效

#### 问题2: TOC 层次缩进
**原因**: 后端 NCX/nav.xhtml 解析拍平了层级
- [x] **NCX 解析**: 递归遍历 navPoint，传入 depth，只取直接子级 `findall("ncx:navPoint", NS)`（而非 `".//"`）
- [x] **nav.xhtml 解析**: 递归遍历 `<ol><li>` 结构，保留嵌套层级
- [x] **spine fallback**: depth=0
- [x] 前端渲染时 `padding-left: 16 + depth * 18px`

#### 问题3: 无链接条目显示
**原因**: 有的 TOC 条目是纯分类标题（无 `<ncx:content>` 或 `href` 为空）
- [x] 后端无 href 也返回，`href=""`
- [x] NCX 解析: `href.split("#")[0]` 去掉锚点
- [x] nav.xhtml 解析: 已 `.split("#")[0]` 保留
- [x] 前端渲染为 `.epub-toc-item.disabled` — 灰色斜体、不可点击、hover 无效果
- [x] Prev/Next 导航自动跳过无链接条目

### v6 中文双语翻译修复（2026-06-03 第六次修改）
