# 前端改版计划 v2

## 设计原则

- **保留现有配色**：当前暖色调（`#FCF9F5` 米白底、`#C8956C` 暖棕强调色、`#2D2D2D` 深灰文字）→ 文艺、书卷气、不侵权
- **布局参考 oTranslator**：首页 Landing + 翻译页双栏阅读器
- **后端零改动**：只改前端 HTML/CSS/JS

## 一、颜色主题（沿用现有，微调）

保持不变。这个色板是自己设计的，与图书翻译主题契合：

| 角色 | 色值 | 用途 |
|------|------|------|
| 背景 | `#FCF9F5` | 页面底色（暖米白） |
| 卡片 | `#FFFFFF` | 卡片、面板 |
| 主文字 | `#2D2D2D` | 标题、正文 |
| 次文字 | `#8A8A8A` | 副标题、描述 |
| 强调色 | `#C8956C` | 按钮、链接 |
| 强调悬停 | `#B8845B` | 按钮 hover |
| 浅强调 | `#F5EDE6` | 标签、浅底 |
| 边框 | `#EDE8E2` | 分隔线 |
| 绿色 | `#5DAA7E` | 下载/完成 |
| 红色 | `#E85D5D` | 停止/错误 |

## 六、后端路由变更

```python

@app.get("/translate")  → translate.html # 翻译页（原 index.html 改）
@app.get("/guide")      → guide.html    # 不变
@app.get("/login")      → login.html    # 预留
```

**API 路径全部不变**（`/api/upload`、`/api/start`、`/api/download` 等）。
