User Guide

Learn how to translate your EPUB books from English to Chinese

Step 1
Install Python
Download and install Python 3.10 or later from the official website. Make sure to check Add to PATH during installation.
Download: python.org/downloads
Step 2
Get Your API Key
Visit DeepSeek Platform to create an API Key. This key is used to call the translation service.
Apply: platform.deepseek.com/api_keys

Screenshot: DeepSeek API Key page

Step 3
Start the Application
Double-click 启动.bat (Windows) or run the command below. The browser will open automatically.
python -m uvicorn server.server:app --host 127.0.0.1 --port 8080
The startup script auto-installs dependencies on first run
Step 4
Configure API Key
On first launch, a settings popup will appear automatically. Enter your API Key and click Save. The key is stored encrypted.

Screenshot: Settings popup with API Key input

Step 5
Open an EPUB File
Click Open a New Book or drag and drop an .epub file onto the page. The book info (title, author, size) will be displayed.

Screenshot: Book info card after upload

Step 6
Start Translation
Click the Start Translation button. The progress bar at the bottom shows real-time translation status.

Screenshot: Progress bar during translation

Step 7
Download the Result
When translation completes, click the Download button to save the translated EPUB file.

Screenshot: Download button after completion

Translation Modes
🌐 Bilingual (default): Keeps original English text and adds Chinese translation below each paragraph. Great for language learning.

🇨🇳 Chinese Only: Replaces English text with Chinese translation. Produces a pure Chinese EPUB.
Security
🔒 Encrypted API Key: Your API key is encrypted with AES-128 before storage.
🗑️ Auto Cleanup: Uploaded files are automatically deleted after translation.
🏠 Local Processing: Everything runs on your machine. No data sent to third parties.

FAQ

What API does this tool use?
By default it uses DeepSeek API. You can switch to any OpenAI-compatible API by changing the API Base URL and Model in settings.
What file formats are supported?
Only .epub files are supported. The tool preserves all formatting, images, links, and table of contents.
Can I interrupt and resume translation?
Yes. Click Stop to pause. Translated content is cached, so resuming continues from where you left off.
Why is my API Key showing as invalid?
Check that your key is correct and your account has sufficient balance. You can regenerate a key at DeepSeek Platform.
Will code blocks be translated?
No. Code blocks (<pre>, <code>) are automatically detected and skipped. Only comments within code blocks may be translated.
